square-biomes is a Fabric mod for Minecraft 1.21.11 that replaces natural biome borders with a strict square grid.

Each biome is resolved per square cell, producing a clean tiled overworld instead of vanilla's curved climate-driven layout. The mod also supports strong randomization, so overworld biome distribution can be much less dependent on vanilla climate rules.

Features

- Square biome borders
- Configurable biome square size
- Configurable cell jitter
- Configurable random biome chance
- Random biome mode applies only to the overworld
- Nether and End are excluded from random biome selection
- Uniform random selection across unique overworld biomes

Default config

- gridSizeBlocks: 128
- cellJitter: 0.85
- randomBiomeChance: 0.8

Requirements

- Minecraft 1.21.11
- Fabric Loader
- Fabric API
- Java 21

Important

- New worlds are recommended
- Existing worlds can show visible borders where new chunks generate
